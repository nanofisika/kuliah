# kuliah

Repositori publik media pembelajaran, video perkuliahan, dan kode sumber komputasi saintifik dari kanal YouTube [Nanofisika](https://www.youtube.com/@nanofisika).

Kanal dan repositori ini dikelola oleh **Samarthya Lykamanuella** ([@samarlyka](https://github.com/samarlyka)), mahasiswa S1 Fisika dari Fakultas Sains dan Matematika (FSM) Universitas Kristen Satya Wacana (UKSW), Salatiga.

## Lisensi

Repositori ini dilisensi menggunakan dua lisensi:

- **Apache 2.0** untuk skrip-skrip dan kode-kode sumber program (seperti skrip Python, Batchfile, dsb.)
- **CC BY-NC-SA 4.0** untuk file-file lainnya (seperti PPT, gambar, dsb.)

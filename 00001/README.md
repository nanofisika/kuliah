## Deskripsi Singkat

Folder ini berisi contoh file spesifikasi MIF berupa simulasi keadaan dasar (_ground state_) dan juga simulasi dengan medan luar dari kasus _spinvalve_ berjenis _giant magnetoresistance_ (GMR). 

- **`spinvalve_gmr_gs.mif`:** simulasi GMR keadaan dasar yang menghasilkan file OMF, yang bisa dimasukkan ke dalam spesifikasi `m0` dari simulasi dengan medan luar.
- **`spinvalve_gmr_medan.mif`:** simulasi GMR dengan medan luar, yang membutuhkan file OMF hasil simulasi keadaan dasar.
